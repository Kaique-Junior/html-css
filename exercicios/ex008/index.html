Principais Formatações

Negrito / Destaque

Colocamos um termo em negrito nessa frase, usando a tag "b"(não semantica)

Agora colocaremos o negrito, usando da tag "strong" (semântica)

Itálico / Ênfase

Colocamos um termo em itálico nessa frase, usando a tag "i"(não semantica)

Agora colocaremos o itálico, usando da tag "em" (semântica)

Texto Marcado

Podemos criar um texto marcado usando a tag "MARK"

Texto Grande e Pequeno

Criando um texto grande e um texto pequeno neste parágrafo.

Texto Deletado

Podemos marcar um texto como exluído para indicar que deve ser lido, mas não considerado utilizando da tag "del"

Texto inserido ou Sublinhado

Podemos indicar que um texto foi inserido usando a tag "ins".

Existe também a tag "u" (não semântica) para indicar que o texto foi inserido.

Subscrito e Sobrescrito

Colocamos um termo em H2O, usando a tag "sub"

Colocamos um termo em E = mc2, usando a tag "sup"
